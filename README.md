# Stone Paper Scissor Game

This is a simple console-based Java game where you play Stone, Paper, Scissor against the computer.

## Features

- Command-line gameplay
- Random computer choices
- Input validation for non-number and out-of-range values
- First to 3 wins becomes the champion

## Requirements

- Java Development Kit (JDK) installed

## How to Run

Compile the program:

```bash
javac StonePaperScissorGame.java
```

Run the program:

```bash
java StonePaperScissorGame
```

## How to Play

- Enter `1` for `Stone`
- Enter `2` for `Paper`
- Enter `3` for `Scissor`
- The computer picks a random option each round
- The game continues until either you or the computer reaches 3 wins
